¿Cómo funciona el JavaScript?

Hemos utilizado JavaScript para hacer interactiva la calculadora:

// 1. Obtener elementos del DOM
const num1Input = document.getElementById('numero1');
const num2Input = document.getElementById('numero2');
const calcularBtn = document.getElementById('calcular');
const resultadoDiv = document.getElementById('resultado');

// 2. Agregar evento al botón
calcularBtn.addEventListener('click', function() {
  // 3. Obtener los valores de los inputs
  const num1 = parseFloat(num1Input.value) || 0;
  const num2 = parseFloat(num2Input.value) || 0;

  // 4. Calcular la suma
  const suma = num1 + num2;

  // 5. Mostrar el resultado
  resultadoDiv.innerHTML = `${num1} + ${num2} = ${suma}`;
});

Este código realiza lo siguiente:

  • Captura los elementos HTML usando sus IDs
  • Escucha el evento de clic en el botón "Calcular Suma"
  • Obtiene los valores de los inputs y los convierte a números
  • Calcula la suma de ambos números
  • Muestra el resultado en el div de resultados con formato especial
